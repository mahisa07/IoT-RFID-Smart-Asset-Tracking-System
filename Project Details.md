# Project Details

# IoT-Enabled RFID-Based Smart Asset Tracking System

## Project Description

The IoT-Enabled RFID-Based Smart Asset Tracking System is a web-based application developed to automate asset identification, tracking, monitoring, and inventory management using Radio Frequency Identification (RFID) technology and the Internet of Things (IoT). The system enables organizations to monitor the real-time location and status of assets, reducing manual effort and improving operational efficiency.

The application provides a user-friendly dashboard where administrators can add, edit, delete, search, and monitor assets. RFID readers detect RFID tags attached to assets, and the collected data is displayed through an interactive web interface. The system also supports multilingual functionality (English and Tamil) to improve accessibility for different users.

---

# Problem Statement

Many organizations still rely on manual asset tracking methods, which are time-consuming, error-prone, and inefficient. Missing assets, inaccurate inventory records, and delayed asset identification lead to operational losses and increased management costs.

This project addresses these challenges by providing an automated RFID and IoT-based solution that offers real-time asset tracking, centralized monitoring, and efficient inventory management.

# Modules

### Login Module
Provides secure authentication for administrators.

### Dashboard Module
Displays total assets, active assets, available assets, scanned assets, date, time, and notifications.

### Asset Management Module
Allows users to add, update, delete, and search assets.

### RFID Scan Module
Simulates RFID tag scanning and updates asset status automatically.

### Tracking Module
Tracks asset movements and maintains scan history.

### Reports Module
Generates inventory and asset tracking reports.

### Language Module
Allows users to switch between English and Tamil.
